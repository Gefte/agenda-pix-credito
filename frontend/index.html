2. Dados Coletados

2.1. Dados fornecidos diretamente

  • Dados de identificação: Nome completo, CPF, data de nascimento, RG;
  • Dados de contato: Email, telefone, endereço;
  • Dados profissionais: Histórico profissional, habilidades, certificações;
  • Dados financeiros: Dados bancários, histórico de transações;
  • Dados de acesso: Login, senha, perguntas de segurança.

2.2. Dados coletados automaticamente

  • Dados de uso: Histórico de navegação, interações, serviços contratados;
  • Dados técnicos: Endereço IP, tipo de dispositivo, navegador, localização;
  • Cookies e tecnologias similares: Para melhorar a experiência e segurança.

2.3. Dados de terceiros

Podemos receber dados de parceiros comerciais, redes sociais (quando você conecta suas contas) e serviços de verificação de identidade.

3. Bases Legais e Finalidades

Tratamos seus dados com base nas seguintes bases legais:

3.1. Consentimento

Para finalidades como:

  • Envio de comunicações de marketing;
  • Uso de dados sensíveis;
  • Compartilhamento com terceiros para fins não essenciais.

3.2. Execução de contrato

Para finalidades como:

  • Prestação dos serviços contratados;
  • Processamento de pagamentos;
  • Comunicação sobre a conta e serviços.

3.3. Legítimo interesse

Para finalidades como:

  • Melhoria de nossos serviços e experiência do usuário;
  • Prevenção de fraudes e segurança;
  • Análise de dados para business intelligence.

3.4. Cumprimento de obrigação legal

Para finalidades como:

  • Atendimento a requisições judiciais ou administrativas;
  • Cumprimento de obrigações fiscais e contábeis.

4. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Prestadores de serviços: Processamento de pagamentos, hospedagem, analytics;
  • Parceiros comerciais: Para oferta de serviços complementares;
  • Autoridades competentes: Quando exigido por lei ou processo legal;
  • Outros usuários: Informações limitadas necessárias para transações.

5. Retenção de Dados

Mantemos seus dados pelo período necessário para:

  • Cumprir finalidades para as quais foram coletados;
  • Atender obrigações legais ou regulatórias;
  • Resolver disputas, fazer cumprir nossos acordos.

Após esse período, os dados serão anonimizados ou excluídos de forma segura.

6. Seus Direitos LGPD

Conforme o art. 18 da LGPD, você tem direito a:

  • Confirmação e acesso: Confirmar que seus dados são tratados e acessá-los;
  • Correção: Solicitar correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação: Solicitar a eliminação ou limitação do tratamento;
  • Portabilidade: Solicitar a transferência de dados a outro fornecedor de serviço;
  • Informação sobre compartilhamento: Saber com quais entidades compartilhamos seus dados;
  • Revogação do consentimento: Revogar consentimento a qualquer momento.

7. Segurança de Dados

Implementamos medidas técnicas e organizacionais robustas para proteger seus dados:

  • Criptografia de dados em repouso e em trânsito;
  • Controle de acesso baseado em função (RBAC);
  • Monitoramento de segurança 24/7;
  • Backups regulares e testados;
  • Programa de gestão de vulnerabilidades.

8. Cookies e Tecnologias Similares

Utilizamos cookies para:

  • Funcionamento essencial da plataforma;
  • Análise de uso e desempenho;
  • Personalização de conteúdo e anúncios.

Você pode gerenciar suas preferências de cookies através das configurações do seu navegador.

9. Menores de Idade

Nossos serviços não são direcionados a menores de 18 anos. Se tomarmos conhecimento de que coletamos dados de menores sem consentimento parental, tomaremos medidas para remover essas informações.
